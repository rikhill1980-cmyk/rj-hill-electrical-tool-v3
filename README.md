# R J Hill Homes Electrical Review Tool

This is a deploy-ready Vite + React bundle for your iPad electrical markup workflow.

## Quick deploy to Vercel

1. Create a GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel and import the GitHub repository.
4. Click Deploy.
5. Open the deployed URL on your iPad in Safari.
6. Tap Share > Add to Home Screen.

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Upload JPG or PNG plans.
- Use Print A3 Pack to generate a print-ready browser sheet.
- Use Export Variations for a plain-text approval summary.
